# SAORI PORTFOLIO

個人制作・学習の成果をまとめた、ポートフォリオサイトです。  
HTML / CSS を中心に、レイアウトと構造の整理を目的として制作しています。

## Overview
本リポジトリは、ポートフォリオサイトの実装コード一式を管理するためのものです。  
構造・動作・レスポンシブ時の破綻を起こしにくい実装を前提として構成しています。

## Features
- 横スクロールと縦スクロールを切り替えるショーケース構成
- PCでは Scroll Snap を使用し、モバイル環境では無効化
- 固定フッターを含む UI において、コンテンツとの干渉を回避するレイアウト設計
- 最小表示幅を考慮したレスポンシブ対応

## Tech Stack
- HTML
- CSS（Flexbox / Grid / Scroll Snap）
- JavaScript（Vanilla）

## Notes
本ポートフォリオは、実務を想定した設計練習・検証を目的とした個人制作です。
